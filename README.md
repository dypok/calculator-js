## Características Principales
**Interfaz de Fórmula Abierta:** La "pantalla" está dividida en tres campos independientes y un área de resultado, permitiendo ver todos los componentes de la operación simultáneamente.

**Lógica Acumulativa:** El resultado de una operación se transfiere automáticamente al primer campo para permitir cálculos continuos sin necesidad de reescribir valores.

**Seguridad de Entrada:** Los campos de texto están configurados como readonly (solo lectura), garantizando que la interacción se realice exclusivamente a través de los botones de la interfaz.

**Diseño Responsivo:** Estética moderna con modo oscuro ("Dark Mode") optimizada para una visualización clara.

**Validación de Datos:** Incluye control básico para prevenir errores comunes, como la división por cero.

## Tecnologías Utilizadas
**HTML5:** Estructura de los elementos de entrada y contenedores.

**CSS3:** Estilizado mediante Flexbox para la alineación de la pantalla y Grid Layout para la distribución del teclado.

**JavaScript:** Lógica de programación simplificada mediante el uso de condicionales if y manipulación directa del DOM (Document Object Model).

## Instrucciones de Uso
**Ingreso del primer número:** Utiliza el teclado numérico para escribir en la primera sección.

**Selección de operación:** Pulsa sobre cualquiera de los operadores aritméticos (+, -, *, /).

**Ingreso del segundo número:** Escribe el valor que completará la operación.

**Cálculo:** Presiona el botón CALCULAR. El resultado aparecerá en la última sección y el primer campo se actualizará con este valor para permitir un nuevo cálculo acumulativo.

**Limpiar:** Usa el botón C para reiniciar todos los campos a cero.

## Ejecución
No requiere instalación de dependencias externas. Basta con abrir el archivo .html en cualquier navegador web moderno.
